# File Handling

## Use readfile() to read a file and write it to the output buffer

Code: 

```html
<!DOCTYPE html>
<html>
    <body>
        <?php
            echo readfile("webdictionary.txt");
        ?>
    </body>
</html>
```

Output:

[Click here!](https://tryphp.w3schools.com/showphp.php?filename=demo_file_readfile)
