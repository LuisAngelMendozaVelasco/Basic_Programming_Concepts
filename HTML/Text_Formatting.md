# Text Formatting

## - Bold formatting using the `<b>` element -

Code:

```html
<!DOCTYPE html>
<html>
    <body>
        <p>This text is normal.</p>
        <p><b>This text is bold.</b></p>
    </body>
</html>
```

Output:

<!DOCTYPE html>
<html>
    <body>
        <p>This text is normal.</p>
        <p><b>This text is bold.</b></p>
    </body>
</html>
