# Output

## Use cout to output values/print text

Code:

```cpp
#include <iostream>
using namespace std;

int main() {
    cout << "Hello World!" << endl;

    return 0;
}
```

Output:

```text
Hello World!
```

## Using many cout objects

Code:

```cpp
#include <iostream>
using namespace std;

int main() {
    cout << "Hello World!" << endl;
    cout << "I am learning C++" << endl;

    return 0;
}
```

Output:

```text
Hello World!
I am learning C++
```

## Insert a new line with \n

Code:

```cpp
#include <iostream>
using namespace std;

int main() {
    cout << "Hello World! \n";
    cout << "I am learning C++" << endl;

    return 0;
}
```

Output:

```text
Hello World!
I am learning C++
```
