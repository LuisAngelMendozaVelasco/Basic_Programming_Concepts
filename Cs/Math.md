# Math

## Math.Max(x,y) - return the highest value of x and y

Code:

```csharp
using System;

namespace MyApplication
{
    class Program
    {
        static void Main(string[] args)
        {
            Console.WriteLine(Math.Max(5, 10));           
        }
    }
}
```

Output:

```text
10
```
